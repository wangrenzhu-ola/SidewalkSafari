# TankTheater Care Scenes

Native SwiftUI iOS app for beginner aquarists to create, save, compare, and revisit local Care Scenes. The first one-tank care loop stays free; Premium is additive for multiple tanks, longer history, export, and extra stage themes.

- Platform: iOS SwiftUI / Swift
- Locale: en-US
- Persistence: local JSON documents + UserDefaults entitlement/privacy choices
- Privacy: local-first, optional AI note cleanup is user-confirmed and never autonomous
